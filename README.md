 BMW Info Website
BMW Info is a modern multi-page website that showcases BMW vehicles, history, and brand information using a clean luxury automotive design.It is built using HTML, CSS, Bootstrap, and JavaScript.

 Live Pages
 Home Page – Introduction to BMW Info
 Models Page – BMW vehicle showcase
 About Page – Brand mission and identity
 History Page – BMW timeline (1916 → present)
 Contact Page – Contact form & details
 Feedback Page – User feedback system
 
 Features
 Fully responsive design (mobile + desktop)
 BMW-inspired dark luxury UI
 Bootstrap 4 grid system
 Image-based model and history sections
 Hover animations and interactive cards
 Multi-page navigation system

How to Run
Download the project folder
Open index.html in any browser
Use the navigation bar to explore pages
Enjoy the BMW experience 

Purpose
This project was built to:
Practice front-end development
Learn multi-page website structure
Improve UI/UX design skills
Build a luxury automotive-themed website
